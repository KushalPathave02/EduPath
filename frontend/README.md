# ISAII Student Welfare App

This is a React Native application built with Expo to provide guidance and support for students.

## Features

- **Quiz**: Helps students discover their interests and potential career paths.
- **Guidance**: Provides personalized roadmaps based on quiz results.
- **Planner**: Allows students to create and manage their study schedules.
- **Progress**: Tracks progress through roadmaps and quizzes.
- **Interviews**: Helps students schedule and prepare for mock interviews.

## Setup and Running the App

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Start the Expo Development Server**:
    ```bash
    npm start
    ```
    This will open the Expo developer tools in your browser.

3.  **Run on a Mobile Device**:
    - Install the **Expo Go** app on your iOS or Android device.
    - Scan the QR code from the Expo developer tools with your device's camera.

## Project Structure

- `App.tsx`: The main entry point of the application.
- `src/navigation/`: Contains the navigation logic (Tab Navigator).
- `src/screens/`: Contains all the main screens of the app.
- `src/components/`: Contains reusable UI components.
- `src/data/`: Contains mock data for the application.
- `src/store/`: Contains state management logic and types.
- `src/utils/`: Contains helper functions.
- `assets/`: Contains app icons and images.
